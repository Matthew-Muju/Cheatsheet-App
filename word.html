Microsoft Word Cheatsheet

Master Word with essential formatting tips, keyboard shortcuts, and step-by-step tutorials

format_align_left Essential Formatting Features

style Styles

Apply predefined formatting to text elements for consistent document appearance.

How to use:

Select text and choose from Styles gallery in Home tab (Heading 1, Normal, etc.)

format_list_numbered Numbering & Bullets

Create organized lists with automatic numbering or bullet points.

How to use:

Select text and click Numbering or Bullets in the Paragraph group on Home tab

table_chart Tables

Insert and format tables to organize data in rows and columns.

How to use:

Go to Insert > Table and select the number of rows and columns

insert_photo Images & Graphics

Add visual elements to enhance your document's appearance.

How to use:

Go to Insert > Pictures to add images from your computer or online

format_shapes Shapes & SmartArt

Create diagrams and visual representations of information.

How to use:

Go to Insert > Shapes or SmartArt to add visual elements

bookmark Headers & Footers

Add consistent information to the top or bottom of each page.

How to use:

Go to Insert > Header or Footer to add page numbers, titles, etc.

keyboard Keyboard Shortcuts

Text Formatting Shortcuts

Shortcut Description
Ctrl + B Apply or remove bold formatting
Ctrl + I Apply or remove italic formatting
Ctrl + U Apply or remove underline
Ctrl + Shift + > Increase font size
Ctrl + Shift + < Decrease font size
Ctrl + D Open Font dialog box

Paragraph Formatting Shortcuts

Shortcut Description
Ctrl + L Align text left
Ctrl + R Align text right
Ctrl + E Center text
Ctrl + J Justify text
Ctrl + M Indent paragraph
Ctrl + Shift + M Remove paragraph indent

Document Navigation Shortcuts

Shortcut Description
Ctrl + Home Move to beginning of document
Ctrl + End Move to end of document
Ctrl + G Open Go To dialog box
F5 Open Find and Replace dialog box
Ctrl + F Find text
Ctrl + H Replace text

Learn how to create personalized letters, envelopes, and labels using mail merge.

1
Go to Mailings > Start Mail Merge > Step-by-Step Mail Merge Wizard
2
Select document type (letters, envelopes, labels)
3
Connect to your data source (Excel, Outlook, etc.)
4
Complete the merge

Creating Automatic Table of Contents

Generate a professional table of contents that updates automatically.

1
Apply heading styles to your document sections
2
Place cursor where you want the table of contents
3
Go to References > Table of Contents > Automatic Table
4
Right-click and select Update Field to refresh

Using Track Changes for Collaboration

Track edits, comments, and suggestions when collaborating on documents.

1
Go to Review > Track Changes to turn on tracking
2
Make edits - they will be marked with formatting
3
Use Comments to add notes without changing text
4
Use Accept/Reject to finalize changes
