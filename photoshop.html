Adobe Photoshop Cheatsheet

Master Photoshop with essential tools, keyboard shortcuts, and step-by-step tutorials

build Essential Tools

crop Crop Tool

Trim or extend the canvas size of your image.

How to use:

Select the Crop Tool (C), drag to define the crop area, and press Enter to apply

brush Brush Tool

Paint on your image with customizable brushes.

How to use:

Select Brush Tool (B), choose brush settings, and click/drag to paint

format_color_fill Paint Bucket

Fill areas with a solid color or pattern.

How to use:

Select Paint Bucket Tool (G), set fill options, and click to fill an area

content_cut Selection Tools

Select specific parts of your image for editing.

How to use:

Use Marquee (M), Lasso (L), or Quick Selection (W) tools to create selections

healing Healing Tools

Remove imperfections and unwanted elements.

How to use:

Use Spot Healing Brush (J) or Clone Stamp (S) or Clone Stamp (S) to remove blemishes

text_fields Type Tool

Add and edit text in your images.

How to use:

Select Type Tool (T), click where you want to add text

transform Transform Tools

Scale, rotate, and distort objects and paths.

How to use:

Select object, use Free Transform Tool (E) or go to Object > Transform for precise controls
