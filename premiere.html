Adobe Premiere Pro Cheatsheet

Master Premiere Pro with essential editing tools, keyboard shortcuts, and step-by-step tutorials

build Essential Editing Tools

content_cut Razor Tool

Split clips at specific points for precise editing.

How to use:

Select Razor Tool (C), click on a clip to split it, then switch back to Selection Tool (V)

speed Rate Stretch Tool

Change the speed and duration of clips.

How to use:

Select Rate Stretch Tool (R), drag the edge of a clip to speed up or slow down

compare Ripple Edit Tool

Trim clips and close gaps automatically.

How to use:

Select Ripple Edit Tool (B), drag the edge of a clip to trim and shift following clips

slideshow Slip Tool

Change the in and out points of a clip without affecting duration.

How to use:

Select Slip Tool (Y), click and drag on a clip to change its content

tune Lumetri Color Panel

Professional color correction and grading tools.

How to use:

Window > Workspaces > Color, then use Lumetri Color panel to adjust colors

auto_fix_high Essential Graphics Panel

Create titles, lower thirds, and motion graphics.

How to use:

Window > Essential Graphics, then browse templates or create your own graphics

keyboard Keyboard Shortcuts

Playback Shortcuts

Shortcut Description
Space Play/Pause
L Play forward (press multiple times to increase speed)
J Play backward (press multiple times to increase speed)
K Pause
← / → Frame by frame navigation
Shift + ← / → Go to previous/next edit point

Editing Shortcuts

Shortcut Description
C Razor Tool
V Selection Tool
B Ripple Edit Tool
N Rolling Edit Tool
Y Slip Tool
Ctrl + K Add edit at playhead

Timeline Shortcuts

Shortcut Description
+ / - Zoom in/out on timeline
Shift + + / - Zoom in/out on timeline and tracks
\ Maximize or restore active panel
Ctrl + D Default transition
Ctrl + Shift + D Default audio transition
Up / Down Navigate tracks
