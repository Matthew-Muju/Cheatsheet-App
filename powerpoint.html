Microsoft PowerPoint Cheatsheet

Master PowerPoint with essential presentation tips, keyboard shortcuts, and step-by-step tutorials

lightbulb Essential Presentation Tips

format_paint Slide Design

Create visually appealing slides with consistent design elements.

Best practices:

Use consistent fonts, limit text to 6 lines per slide, and apply the 10-20-30 rule (10 slides, 20 minutes, 30pt font)

animation Animations & Transitions

Add movement to your presentation to engage your audience.

How to use:

Select object > Animations > Choose animation effect

Use subtle animations that enhance rather than distract

insert_chart Charts & Graphs

Visualize data effectively with charts and graphs.

How to use:

Insert > Chart > Select chart type

Keep charts simple and label all elements clearly

view_carousel Slide Master

Create consistent layouts and formatting across all slides.

How to use:

View > Slide Master > Edit the master slide to apply changes to all slides

speaker_notes Speaker Notes

Add notes to help you remember key points during presentation.

How to use:

Click in the Notes pane below each slide to add speaker notes

Use Presenter View to see notes while presenting

photo_size_select_actual Images & Media

Enhance your presentation with relevant images and media.

How to use:

Insert > Pictures or Online Pictures

Use high-quality images that support your message

Navigation Shortcuts

Shortcut Description
F5 Start presentation from beginning
Shift + F5 Start presentation from current slide
Esc Exit presentation mode
Page Down / Space Next slide or animation
Page Up Previous slide
B Black screen during presentation

Slide Management Shortcuts

Shortcut Description
Ctrl + M Insert new slide
Ctrl + D Duplicate current slide
Ctrl + Shift + D Duplicate selected object
Delete Delete selected slide
Ctrl + K Insert hyperlink
Ctrl + Shift + D Duplicate selected object
F6 Cycle between panes

Formatting Shortcuts

Shortcut Description

Ctrl + E Center align text
