Adobe After Effects Cheatsheet

Master After Effects with essential animation tools, keyboard shortcuts, and step-by-step tutorials

build Essential Animation Tools

motion_photos_on Keyframes

Create animations by setting keyframes at different points in time.

How to use:

Select a property, click the stopwatch icon to create a keyframe, then move the timeline and adjust values

tune Graph Editor

Fine-tune animation curves for more natural movement.

How to use:

Select keyframes, click the graph editor button, and adjust the curves to control acceleration

layers Masks

Create selective areas to apply effects or hide parts of layers.

How to use:

Select a layer, use the Pen or Shape tools to create a mask, then adjust mask properties

blur_on Effects

Apply visual effects to enhance your animations.

How to use:

Right-click a layer > Effect, or use the Effects & Presets panel to browse and apply effects

auto_awesome Expressions

Create complex animations with JavaScript-based expressions.

How to use:

Alt-click the stopwatch of a property to open the expression editor and write code

view_in_ar 3D Layers

Create depth and perspective with 3D layers and cameras.

How to use:

Enable 3D layer switch, adjust position in 3D space, and add a camera layer for perspective

keyboard Keyboard Shortcuts

Playback & Navigation

Shortcut Description
Space Play/Pause
0 on numeric keypad RAM Preview
. on numeric keypad Preview only current frame
J / K / L Previous frame / Pause / Next frame
Home / End Go to beginning / end of composition
Page Up / Page Down Go to previous / next keyframe

Layer & Composition

Shortcut Description
Ctrl + N New Composition
Ctrl + Y New Solid Layer
Ctrl + Alt + Y New Adjustment Layer
Ctrl + D Duplicate Layer
Ctrl + Shift + D Duplicate Layer with Properties
U Show/Hide all keyframed properties

Animation & Keyframes

Shortcut Description

F9 Easy Ease keyframe

Ctrl + Shift + F9 Easy Ease Out
